# TP_Organizacion_Ventas
Trabajo practico Gestión Colaborativa, Control de Versiones y  Organización Empresarial
Integrantes:
Flores Gustavo, Comision:
Flores Gonzalo, Comision: 
Escenario Elegido: Analisis de ventas de una pequeña empresa.
Descripcion: Con este programa se puede tener informacion precisa de que productos son los mas vendidos, realizando un calculo de totales que permiten a la empresa tener un control del dinero ganado.
DataSet utilizado: Ya que el data set indicado en el PDF carece de toda la informacion a procesar, generamos uno nuevo que logre identificar el producto vendido, este mismo se puede encontrar en la carpeta datos del Proyecto.
Instrucciones Basicas: El mismo programa toma el archivo indicado en la carpeta Datos, en caso de querer alterar la informacion se debe modificar de forma manual el archivo CSV.
